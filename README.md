# LockScript
LockScript — это десктопное приложение для создания и обмена зашифрованными документами.
Все файлы защищены на уровне содержимого: без знания встроенного ключа их невозможно прочитать.
Поддерживается три уровня шифрования, отправка по локальной сети, тёмная/светлая тема, горячие клавиши и многое другое.

🔐 Возможности
Создание защищённых документов с расширением .lscript

Три уровня шифрования:
Простое – самый быстрый тип шифрования
Стандартное – современный надёжный стандарт
Расширенное – улучшенная защита для самых ценных данных

Вкладки — работа с несколькими документами одновременно
Поиск и замена по тексту
Отправка документов по локальной сети (в том числе через Hamachi, Radmin VPN и аналоги)
Приём документов от других пользователей LockScript — файл открывается в новой вкладке
Тёмная и светлая темы оформления
Горячие клавиши для всех основных действий (Ctrl+N, Ctrl+S, Ctrl+F и др.)
Меню недавних файлов для быстрого доступа
Экспорт на печать через системный диалог

📥 Установка и запуск
Скачайте последнюю версию LockScript.exe из раздела Releases
Запустите файл — приложение готово к работе

Системные требования:
Windows 10 / 11 (64-бит)

Для передачи файлов по сети может потребоваться разрешить доступ в брандмауэре
При использовании виртуальных частных сетей (Hamachi, Radmin VPN) передача работает между любыми компьютерами внутри сети

🚀 Быстрый старт
Создайте новый документ (Ctrl+N) или откройте существующий (.lscript или .txt)

Напишите текст
Нажмите «Сохранить» (Ctrl+S) — при первом сохранении выберите уровень шифрования
Чтобы отправить файл:
Нажмите кнопку 📡 Сеть → введите send
Укажите IP-адрес получателя и порт (по умолчанию 55555)
Получатель должен нажать 📡 Сеть → ввести receive и тот же порт
Горячие клавиши показаны в меню и по нажатию F1

🛡️ О безопасности
Парольная фраза жёстко вшита в программу и не запрашивается у пользователя — это сделано для максимального удобства при обмене документами между доверенными участниками.

Без знания этой фразы (и, соответственно, без программы) расшифровать файл практически невозможно.

Файлы .lscript зашифрованы и не читаются в текстовых редакторах.

При передаче по сети документ также передаётся в зашифрованном виде, что исключает перехват в открытом виде.
